Impinj Reader Chip API¶

The Impinj RAIN RFID reader chips (“Impinj reader chips”) have an integrated Cortex-M0+ microcontroller with embedded RAIN firmware providing the GS1 UHF Gen2 RAIN protocol as a pre-integrated feature. A host processor communicates with the Impinj reader chip using the Impinj reader reader chip host API. The Impinj reader chip host API provides a consistent interface across the entire Impinj reader chip family of products including the Impinj E710, Impinj E510 and Impinj E310. The Impinj reader chip host API is comprised of 5 components:

Wireline

Message framing and flow control

Messages

The set of messages transported over the wireline

Address Map

Most of the configuration and status is accessed in this address map via the Read and Write messages

Operations

These are the actions that can be done with the Impinj reader chip. They are initiated via writes to the address map and will generate interrupts to the host when complete. Some operations will produce data in the EventFifo.

FIFOs

Some data are streamed back to the user in FIFOs, for example tag reports.
